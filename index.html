Steganography Website

For UW-Stout's software engineering principles class, I and a group of five others worked together to make a steganography application using the agile development approach. During the project, I was the scrum-master in charge of the development process. Our application was a web-based application for steganography using the least significant bit method.
Steganography is the process of encoding information into image files. The least significant bit method is when individual bits are encoded into the colors of the pixels themselves. You can read more about it in our project's read me file. As an example of what steganography can do, look at the portrait of William Shakespeare. Using our application, we were able to hide the entire works of William Shakespeare inside that PNG file.
You can visit our steganography website here, it's hosted statically with github pages.

Wi-Fi Heatmap for Android

This project was my final project for UW-Stout's Mobile Applications Development for Android class. The purpose of the application was meant to be a mixture of differing technologies merged together to make a useful application. It serves to map the strength of a Wi-Fi network and displays it as an overlay over a Google Map. The data is collected automatically for the user using native Android utilities and stored locally on the device using a Room Database. The group consisted of four students, including myself, and was concluded in the course of a month. My primary focus was creating a GUI that allowed the user to manipulate the database in a Model-View-ViewModel design pattern, and once that was complete I spent the rest of the project focused on GUI improvments.

Exploration of the RSA encryption

The RSA encryption system is a very popular encryption system because it allows two computers who have never communicated with each other before to make a secure, encrypted connection. Based in number theory, the RSA uses the concepts of modular exponentiation and properties of prime numbers to turn your information into extremely large integers. To give an example of how large these numbers are, an RSA 1024 bit encryption will create numbers on the power of 10309, which is one followed by over three hundred zeros. I created this program not to actually encrypt data (my project should not be used to encrypt any data for security purposes) but rather to explore the concepts of prime cryptography as explained here.

Emergence Theory Proof of Concept

Emergence theory is the scientific concept that a large group of objects is greater then the sum of its parts. For example: a single water molecule doesn't make a floor slippery, but a large quantity of water molecules will make a slippery puddle. Being slippery is a qualitative property of water that only occured when a lot of water molecules came together. In other words, a new property emerged. I first learned the basic principles of emergence theory in a Kurzgesagt video on YouTube. Over a weekend, I decided to model the emergence theory concept on a computer using random number generators to replicate a rudimentary version of ant colony pheromone behaviors.
I started with a conceptual design and moved onto the actual code in a simple console application made with Microsoft Visual studio. This project was interesting to me not because of the complexity of the task, but because of the application of a mathematical concept.
